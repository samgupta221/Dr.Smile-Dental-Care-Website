# 🦷 Dr. Smile Dental Care Website

A premium, modern, and fully responsive dental clinic website built with **HTML5**, **CSS3**, and **JavaScript**. The website delivers a luxurious user experience with elegant animations, responsive layouts, appointment booking, pricing plans, doctor profiles, testimonials, and FAQs.

---

## 🌐 Live Demo

🔗 **Website:** https://meek-torte-4e61c6.netlify.app/

## ✨ Features

- 🦷 Premium Dental Clinic Landing Page
- 📱 Fully Responsive Design
- 🎨 Modern Luxury UI/UX
- 📅 Appointment Booking Form
- 👨‍⚕️ Doctor Team Section
- 💰 Pricing Plans
- ⭐ Patient Testimonials
- ❓ Interactive FAQ Section
- 📍 Contact & Clinic Information
